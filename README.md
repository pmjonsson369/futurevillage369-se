# 369 Future Village

En premium skandinavisk webbplats för trädgårdsdesign och hållbart boende.

## Om projektet

369 Future Village erbjuder tjänster inom smart trädgårdsplanering, villadesign och ekobyar – med fokus på funktion, hållbarhet och skandinavisk minimalism.

### Tjänster

- **Smart Garden** – Genomtänkt planering av trädgård, odling och skötsel
- **Smart Garden Analysis** – Personlig trädgårdsanalys med konkret plan (intropris 1 499 kr)
- **Smart Growing Systems** – Smarta odlingslösningar för hem och trädgård
- **Smart Greenhouse** – Intelligenta växthus med integrerade system
- **Smart Villa** – Framtidens boende där trädgård, vatten och energi samverkar
- **Ecovillage** – Hållbart boende med fokus på helhet och gemenskap

## Teknikstack

- **React 18** med TypeScript
- **Vite 5** som byggverktyg
- **Tailwind CSS v3** för styling
- **shadcn/ui** för UI-komponenter
- **React Router** för navigering

## Kom igång

```bash
npm install
npm run dev
```
